# Zorvyn Finance Dashboard 🚀

A premium, high-performance personal finance tracker built with **React**, **Tailwind CSS**, and **Zustand**. Featuring a sleek dark mode, real-time analytics, and persistent data storage.

## ✨ Features
* **Dynamic Dashboard:** Real-time insights into spending vs. income.
* **Persistent State:** Data stays saved even after refresh using Zustand Middleware + LocalStorage.
* **Animated UI:** Smooth transitions and micro-interactions powered by Framer Motion.
* **Role-Based View:** Toggle between "Admin" and "User" modes to see different dashboard perspectives.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.

## 🛠️ Tech Stack
* **Frontend:** React 18, Vite
* **Styling:** Tailwind CSS, Styled-Components (for animated switches)
* **State Management:** Zustand
* **Charts:** Recharts
* **Animations:** Framer Motion
